Semantic roles

Reach for these before raw palette values.

PRIMARY
--sk-primary
Main action. One per screen. Hero CTAs, "START", "INSERT COIN".
ACCENT
--sk-accent
Selected state, hover ring, data viz, secondary highlight.
SUCCESS
--sk-success
Quest complete, validation passed, terminal "OK".
WARNING
--sk-warning
Coins, score, S+ rank, attention without alarm.
DANGER
--sk-danger
HP loss, destructive action, critical error.
INFO
--sk-info
Tooltips, neutral status, system messages.
RING
--sk-ring
Focus outline. 2px solid, offset 2px from element.
BORDER
--sk-border
Default panel edge. Use 1–2px; pixel UIs are square.
